📋 Configuration rapide

Pour que vos équipements apparaissent automatiquement, configurez-les pour envoyer leurs logs à :

Serveur Syslog : ...
Port : 514
Protocole : UDP ou TCP
config log syslogd setting
  set status enable
  set server "192.168.10.105"
  set port 514
end

⚙️ Actions système

📚 Documentation rapide

Commandes utiles :
• Vérifier rsyslog : sudo systemctl status rsyslog
• Logs d'archivage : tail -f /var/log/syslog-archiver.log
• Espace disque : df -h /mnt/syslog-data
• Port 514 : sudo netstat -tulpn | grep 514
• Logs du jour : ls -la /mnt/syslog-data/current/$(date +%Y/%m/%d)/
